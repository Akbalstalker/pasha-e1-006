1. Что такое EcmaScript?

EcmaScript это своего рода спецификация для скриптового языка, которая содержит правила, рекомендации для того, чтоб язык считался совмесным с EcmaScript, при условии соблюдения этих правил и рекомендаций.Ecma 262 являеться стандартом и спецификацией для EcmaScript

2. Для чего используются и чем отличаются атрибуты async и defer?

async загружает скрипт асинхронно. Если скриптов несколько, то первым сработает тот что раньше подгрузился; defer же откладывает выполнение до тех пор, пока страница не будет загружена полностью. При наличии нескольких скриптов загружает их по очереди.

3. Почему предпочтительнее подключать скрипты перед </body>, а не в <head>...</head>?

Потому, что в случае подключения скриптов в конце кода, то есть перед закрывающим тэгом </body>, с начала будет прогружаться содержимое страницы, а затем скрипты. Пользователь увидит в первую очередь содержание страницы. Если же подключать в head, содержимое прогрузиться через какое-то время после скриптов, и пользователь может покинуть сайт, либо посмотреть рекламу.

4. Какие основные отличия интерпретируемого языка программирования от компилируемого?

Компилируемые языки преобразуются в машинный код компилятором, и являются платформо-зависимыми. Они являются инструкцией для машины, и не понятны для человека после компиляцыи.
Интерпритируемые - выполняются построчно программой-интерпритатором. Считаеться, что скорость выполнения компилируемого языка выше.

5. Что такое языки высокого и низкого уровня? Какие ты знаешь?

Языки низкого уровня по своему содержанию близки к машинному коду, но код обычно длинный и сложный для понимания. Примеры: Assembler, С ,С++. Дают прямой доступ к памяти. Высокоуровневые языки напротив, далеки от машинного кода, компактны и просты для понимания и поиска ошибок. Высокое абстрагиравание. Пример: Python, Ruby, JavaScript, Common Lisp.

6. Для чего используются переменные? Какие ключевые слова используются для создания переменных? Какие есть отличия в переопределении значений при задании переменной через let, const и var?

Переменные используються для записи информации в выделенной области памяти, и последующем доступе к этой информации с помощбю имени переменной.
Для создания переменных используються ключевые слова: bolean, break, long, class, export, import, и др. Они не могут быть именем переменной.
Переменная, обьявленная через let видна только в рамках обьявленного блока{....}, и только после того, как будет обьявленна; через var - видна везде в функции; const - задает переменную, которую нелзя менять - константу.

7. Может ли имя переменной начинаться с цифры, символов _ или $?

Первый символ не может быть цыфрой, однако символы _ и $ могут быть как в начале имени переменной, так и сами являтся пременной.

8. Какие слова запрещено использовать в названии переменной?

Запрещено использовать зарезервированые ключевые слова, которые используються в js для выполнения определенных функций, такие как: var,class,return,export,while,is,do,for, else, и другие.

9. Что такое camelCase и snake_case?

Способ запси camelCase (верблюжей нотацией), это так называемый стандарт для написания имени переменных, присущий именно javascript. snake_case Это второй способ написания переменных, в которых слова разделяются подчеркиванием"_". Он альтернативный, но более длинный. Легче читается(имхо).

10. Что такое "типы данных"? Какой оператор позволяет определить тип данных?

Тип данных:

  1. Число `number`
  2. Строка `string`
  3. Булевый тип (логический) `boolean`
  4. Специальное значение `null`
  5. Специальное значение `undefined`
  6. Объекты `object`
Тип данных можно определить с помощью оператора typeof. Он возвращает тип переменной в виде строки.

11. Что такое "динамическое приведение типов данных"? Можно ли писать JavaScript со статическим приведением типов?

При динамическом приведении в одну и ту же пременную есть возможность записать значения разных типов, тип переменной будет менятся. JavaScript будет возвращать значение для любого выражения.

13. Чем отличаются операторы "==" и "==="?

Два символа равенства используется для сравнения, один символ = означает присваивание. Три знака означает строгое равенство, если два операнда строго равны, он возвращает true.
