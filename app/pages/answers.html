1. Что такое массив?

Набор элементов, расположенных по порядку, имеющих свой порядковый номер - индекс. Имет дополнительный методы массивов.

2. Чем массив отличается от объекта?

Для обращения к элементу массива используют индекс, который в свою очередь хранится в массиве в виде строки. Однако индексами могут быть только неотрицательные целые числа. При добавлении к массиву элемента с индексом, большим чем длинна массива, автоматически обновляется свойство length. Кроме того массивы имеют дополнительные методы массивов.

3. Что такое callback?

Любая функция, которая передается другой функции в качестве аргумента, и выполняется только после того, как основная функция завершила выполнение.

4. Чем отличаются методы Array.of и Array.from?

Array.from - извлекает из массивоподобного элемента или итерируемого объекта данные и записывает их в новый экземпляр Array Array.from - Создает новый массив Array, в который принимает нограниченное число элементов, вне зависимости от типа данных.

5. Как можно узнать длину массива?

У массивов есть свойство length, с помощью которого можно узнать количество элементов в массиве. Для вычисления длинны массива к индексу последнго элемента прибавляется 1.

6. Почему методы pop/push быстрее, чем shift/unshift?

При использовании методов shift/unshift происходит не только удаление (добавление) элемента с индексом 0, остальные элементы перемещаются на новые индексы.
